Mathematical Concepts

SO(3) Rotations

Special Orthogonal group representing 3D rotations. Essential for camera orientation and robot joint movements.

3D Rotations Orthogonal Matrices

SE(3) Transformations

Special Euclidean group for rigid body motions combining rotation and translation in 3D space.

Rigid Body Motion Twists

Lie Algebras

Tangent spaces of Lie groups, providing linear approximations for optimization and velocity representations.

Tangent Spaces Linear Algebra

Pose Graph Optimization

Global optimization technique for SLAM that corrects accumulated drift through loop closures.

Get Started

1

Install Dependencies

Use the modern uv package manager for fast dependency resolution.

uv sync
2

Run Your First Animation

Start with a simple SO(3) rotation visualization.

uv run manim -pql so3_visualization.py SO3RotationVisualization
3

Explore and Customize

Modify parameters in config.py and create your own animations.

# Customize in config.py
ANIMATION.default_wait_time = 1.5
